# 快速开始 - 双图输出功能

## 立即体验

运行以下命令启动交互式可视化工具：

```bash
python3.13 fibonacci_cli_visualizer.py
```

## 简单示例（模=50）

按照以下步骤操作：

```
请选择 (0-5): 1
请输入模数 (1-1000000): 50

请选择 (0-5): 2
请输入: all

请选择 (0-5): 3
是否保存图表为PNG文件? (y/n): y
```

完成后，你会得到两张图片：

1. **fibonacci_mod_50_visualization.png** - 正常尺寸（约50KB）
2. **fibonacci_mod_50_highres.png** - 高清大图（约300KB，6000×6000像素）

## 查看结果

```bash
# 查看生成的文件
ls -lh *.png

# 打开图片查看
open fibonacci_mod_50_highres.png  # macOS
xdg-open fibonacci_mod_50_highres.png  # Linux
```
